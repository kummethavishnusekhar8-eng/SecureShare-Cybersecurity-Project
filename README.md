# 🔐 SecureShare – Secure File Sharing Platform

SecureShare is a secure file sharing platform built using **Python (Flask)**. It allows users to securely upload, encrypt, manage, preview, and share files with password protection and expiring links.

## Features

- 🔑 User Authentication (Login & Register)
- 📤 Secure File Upload
- 📥 Secure File Download
- 🔒 AES File Encryption
- 👁️ File Preview
- 🔗 Secure Share Links
- 🔐 Password-Protected Sharing
- ⏳ Expiring Share Links
- ⭐ Favorite Files
- 🔍 Search & Sort Files
- 👤 User Profile
- ⚙️ Settings Page
- 📱 Responsive Bootstrap UI

## Technologies Used

- Python
- Flask
- SQLAlchemy
- Flask-Login
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- SQLite
- AES Encryption
