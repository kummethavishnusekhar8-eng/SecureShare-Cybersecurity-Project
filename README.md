SecureShare is a **cybersecurity-focused secure file sharing platform** developed using **Python (Flask)**. It enables users to securely upload, encrypt, manage, preview, and share files while implementing modern security practices such as **AES encryption**, **password-protected sharing**, **expiring links**, and **secure authentication**.
